# BOM Comparison Tool

A web application for engineers and procurement teams to upload and compare Bills of Materials (BOM) across multiple file formats — Excel, CSV, PDF, Word.

**Live:** https://bom-comparison.onrender.com

## Tech Stack

Python · Django 5.2 · PostgreSQL (Neon) · Pandas · OpenPyXL · PDFPlumber · Bootstrap 5 · Render

## Features

- Upload and compare BOM files (Excel, CSV, PDF, DOCX)
- Smart column detection and key-based matching
- Side-by-side diff view: added, removed, changed rows
- Comparison history per user account
- File upload tracking and user profile stats
