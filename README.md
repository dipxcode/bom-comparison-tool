# BOM Comparison Tool

A web-based BOM (Bill of Materials) comparison platform built with **Python, Django, Bootstrap, and PostgreSQL**.

## Overview

The BOM Comparison Tool allows users to upload a **Master BOM** and compare it against up to **5 BOM files** to identify:

* Matching components
* Missing components
* Additional components
* Mismatched data
* Match percentage/score

The platform is designed as a multi-phase project, starting with core comparison capabilities and expanding into advanced analytics and AI-driven BOM intelligence.

---

## Phase 1 Features

### Authentication

* User registration and login
* Secure authentication
* Login/Signup via modal (no separate pages)

### Landing Page

* Product overview
* Key features and workflow
* Call-to-action for users

### Dashboard

* Comparison statistics
* Recent activities
* Uploaded files
* Comparison history

### BOM Comparison

* Upload 1 Master BOM
* Upload up to 5 comparison BOM files
* Support for approved file formats (CSV/XLS/XLSX as per requirements)

### Results

* Tabular comparison view
* Match and mismatch statistics
* Component-level comparison
* Export results in JSON format

### History

* View previous comparisons
* Access uploaded files and generated results
